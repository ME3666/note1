常见 OJ 错误点及解决方案

  1. 字符串结尾要加上 '\0'，不然会输出奇怪字符
    在处理字符串时，确保每个字符串都有一个结尾符 '\0'，否则可能会出现输出异常或程序错误。

  2. 一直过不了的题检查算法问题
    如果一个题目一直无法通过，可能是算法上存在问题。检查算法的时间复杂度和实现逻辑，确保其能够正确处理边界情况。

  3. 可以有三维数组
    C 语言支持使用三维数组。例如，可以声明一个三维数组如下：

    int arr[10][10][10];  // 三维数组

  4. printf("\n"); 中双引号
    在使用 printf 打印换行时，确保使用正确的双引号 " " 包围字符串：

    printf("\n");  // 正确的换行输出

  5. 未赋值的数组输出后是乱码
    如果数组未被初始化或赋值，输出时会出现乱码。务必确保在使用数组前先进行初始化。

  6. 冒泡排序中内层循环，每次开始处不都是 0
    在进行冒泡排序时，内层循环的起始索引应根据外层循环的当前索引来调整。一般来说，内层循环不必从 0 开始，而是从外层循环索引 i+1 开始。

  7. 求一个数的特别高次幂时，除了要用循环来快速幂，还要在过程中的每一步求模
    在计算高次幂时，如果涉及到模运算，应该在每一步计算中都进行求模操作，以避免结果过大：

    result = (result * base) % mod;

  8. qsort：一种排序方式

    qsort 是 C 标准库中的快速排序函数，其声明如下：

    void qsort(void *base, size_t num, size_t size, int (*compar)(const void *, const void *));
    • base：指向要排序的数组的第一个元素。
    • num：数组中元素的个数。
    • size：数组中每个元素的大小（字节数）。
    • compar：比较函数，决定元素的排序方式。该函数返回值为：
      • 小于 0：第一个元素小于第二个元素。
      • 等于 0：两个元素相等。
      • 大于 0：第一个元素大于第二个元素。

  9. 二进制操作

    • x % 2 vs x & 1：

      • x % 2 使用模运算，计算 x 除以 2 的余数，结果为 0 表示偶数，1 表示奇数。
      • x & 1 使用按位与运算，检查 x 的最低有效位（LSB）。如果是奇数，最低有效位为 1；如果是偶数，最低有效位为 0。

    • while (x) { x &= (x - 1); count++; }：
      这个循环用于统计 x 中 1 的个数（即 x 的二进制表示中有多少个 1）。每次通过 x &= (x - 1) 操作去掉 x 的最低有效位 1，然后将计数器 count 增加 1。

    • 交换两个整数的值（不使用临时变量）：

      swap(int *a, int *b) {
    *a = *a ^ *b;  // a 存储了 a 和 b 的异或结果
    *b = *a ^ *b;  // b 变为原来的 a 的值
    *a = *a ^ *b;  // a 变为原来的 b 的值
}

    • for (int i = 0; i < n; i++) { result ^= arr[i]; }：
      这段代码计算数组 arr 中所有元素的按位异或结果。result 会被初始化为 0，然后依次与数组元素进行异或操作。异或操作具有“抵消”性质，即两个相同的数字异或后会得到 0，因此它可以用来查找数组中只出现一次的数字，或者做其他与异或相关的计算。

  10. scanf 不能读取空格
    使用 scanf 读取字符串时，空格会被当作分隔符，导致只读取到第一个空格前的内容。如果需要读取包含空格的字符串，应该使用 fgets 或者 %[^\n] 格式指定符：

    char str[100];
fgets(str, sizeof(str), stdin);  // 读取包含空格的字符串

  11. 循环嵌套时，看好变量初始化条件在哪个循环里
    在嵌套循环中，确保每个变量的初始化操作在合适的地方。例如，如果变量在每次迭代中都需要重置，则应将其初始化放在内层循环中。

  12. 对于长长的数字，可以使用字符串减少时间复杂度
    对于非常大的数字，使用字符串而不是数字类型来处理，可以减少计算的复杂度，尤其是在需要进行大数运算时。

  13. 筛选素数时，对于较大数字使用 for (int i = 2; i <= sqrt(n); i++)
    在筛选素数时，对于较大的数字，检查范围可以缩小到 sqrt(n)，即只需要检查到 n 的平方根。

  14. sqrt 返回 double 类型
    sqrt 函数返回值类型为 double，即使输入参数为 float 或 int 类型，返回值仍为 double。

  15. 字符串常用函数

    • strlen(a);计算字符串的长度（不包含\0）
    • strcpy(a,b);将b复制到a中。
    • strncpy(a,b,n);将b中前n个复制到a中。
    • strcat(a,b);将b拼接到a结尾。
    • strncat(a,b,n);将b中前n个拼接到a末尾。
    • strcmp(a,b);比较两个字符串。如果字符串相等，返回0；如果第一个字符串大于第二个，返回正值；如果第一个字符串小于第二个，返回负值。
    • strncmp(a,b,n);比较两个字符串的前 n 个字符。
    • strncmp(a+m,b,n);比较a中从m开始的n个字符和b是否一样。
    • strchr(a,'x');查找字符串a中第一次出现字符x的位置。
    • strrchr(a,'x');查找字符串a中最后一次出现字符x的位置。
    • strstr(a,"abc");查找abc中字符串abc第一次出现的位置。
    • _strupr(a);将a中字符全变成大写。
    • _strlwr(a);将a中字符全变成小写。

  16. 关于宏的运算
    先将所有宏替换，再按顺序计算。

  17. 动态内存分配
    int *arr = (int *)malloc(5 * sizeof(int));分配一个存储 5 个整数的内存块。
