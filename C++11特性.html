day13-C++11-auto声明

  • auto a = 1.1;可以自动识别a的类型。
  • 但是如果a没有初始赋值，是不可以键盘录入的。
  • 使用迭代器，要调整到C++11之后。

day14-C++11-基于范围的for循环

  • for-each循环
    详细教程点在这里
#include<bits/stdc++.h>
using namespace std;
int main()
{
	vector<string>v(10);
	for (auto &x : v)cin >> x;//使用引用，可以修改内容
	for (auto x : v)cout << x;//不使用引用，只能访问
	return 0;
}

day15-C++11-to_string(把数字转化为字符变量) 

#include<iostream>
#include<string>
using namespace std;
int main()
{
	string s = to_string(123.1);//把数字转化为字符变量
	cout << s << endl;
	printf("%s", s.c_str());
}

day16-C++11-stoi和stod(将字符串转化成其他变量)

  • string to int 和 string to double
#include<iostream>
#include<string>
using namespace std;
int main()
{
	int a = stoi("12");//转换成int
	cout << a << endl;
	double b = stod("3.1");//转换成double
	cout << b << endl;
}
