day00-杂项

  • cin是有返回值对象

    • while (cin >> word) 是一个用于从输入流中读取数据的常见结构
    • 可替换c中的while (scanf("%s", word) != EOF)

  • 替换字符串中某一部分--replace方法

    • str.replace(a, b - a + 1, p); 使用 replace 方法替换(从索引几开始，往后数多少个字符，替换成什么)

  • 字符串反转--swap函数

    • #include <utility>头文件中，有swap(a,b);交换a，b的值
    • 如果你要反转字符串中的某个部分如a到b
      // 反转从 a 到 b 的子串
while (a < b) {
	swap(str[a], str[b]);
	a++;
	b--;
}

  • 乘方和开方--pow和sqrt

    // 计算两点之间的距离
double distance = sqrt(pow(x2 - x1, 2) + pow(y2 - y1, 2));

  • 弧度制

    • M_PI 是 C/C++ 标准库中定义的一个宏，表示圆周率 π
    • sin(),cos()括号中只能是弧度制

  • 设置输出精度

    • 使用cout << fixed << setprecision(n);设置精度，而且设置一次就可以
      #include <iostream>
  #include <iomanip> // 包含 setprecision 和 fixed
  using namespace std;

  int main() {
  	double num1 = 3.14159265358979323846;
  	double num2 = 2.71828182845904523536;

  	// 设置小数位数为 8
  	cout << fixed << setprecision(8);

  	// 输出 num1
  	cout << "num1: " << num1 << endl;

  	// 取消 fixed，恢复默认格式
  	cout << defaultfloat;

  	// 输出 num2（默认格式）
  	cout << "num2: " << num2 << endl;

  	return 0;
  }

  • 二进制运算思路

    • 12瓶水，一瓶是毒药，四只小鼠一次性检验哪瓶是
    • 12的二进制是1100，从第一瓶开始，转换成二进制，0001，0010，0011......
    • 四只小鼠排号位置，那个位置为1的就喝下，对应，比如，小鼠情况是：死死活活，对于1100，那么第12瓶水是毒药

day02-字符串及其常用操作

  • 如果直接拿字符串作比较，比较的是两个字符串的码表，从第一个字母开始，依次比较
  • 取子串
    .substr(起始下标, 子串长度)
  • 查找字符串
    .find(字符串, 起始下标)
  • 尾接字符串一定要用 += 省时间
  • 类型转换
    to_string()
    stoi
    stod
#include<bits/stdc++.h>
using namespace std;

int main(void)
{
	//直接赋值
	string s = "hahahawozhenshuai";
	string s2 = "def";
	//直接相加
	string s3 = s + s2;
	//相关操作
	string s4;
	//cin >> s4;//这种方式只能一次录入一个单词
	getline(cin, s4);
	//打印索引2，3 
	string s_sub = s.substr(2, 3);
	//打印索引3之后的内容
	string s_sub2 = s.substr(3);
	
	cout << s3 << endl;
	cout << s4 << endl;
	cout << s_sub << endl;
	cout << s_sub2 << endl;
	cout << s4.length() << endl;
	return 0;
}
