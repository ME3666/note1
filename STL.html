STL：

  • 以下STL容器的大小，都可以使用size()方法来获取
  • 数组(vector)，集合(set)，键值对(map)都可以用迭代器,键值对是以一种类似于结构体指针的方式。
  • 栈(stack)和队列(queue)不可以用迭代器,栈只可以访问头部，对列可以访问头部和尾部。

day04-STL_vector(数组)

  • 是可以不给初始长度，由键盘录入的
  • 如vector<int>v(n);cin>>n;
  • 二维数组
    • vector<vector<int>> mat(100, vector<int> (666, -1)) 构造初始100行，初始666列的二维数组，初值为-1
  • 尾接 & 尾删
    • .push_back(元素)：在 vector 尾接一个元素，数组长度
    • .pop_back()：删除 vector 尾部的一个元素，数组长度
    • 时间复杂度：均摊 O(1)
  • 获取长度
    • .size()
    • 时间复杂度：O(1)
  • 清空
    • .clear()
    • 时间复杂度：O(n)
  • 判空

    • .empty()

    • 如果是空返回 true 反之返回 false.

    • 时间复杂度：O(1)

  • 改变长度

    • .resize(新长度, [默认值])

    • 修改 vector 的长度

    • 如果是缩短，则删除多余的值

    • 如果是扩大，且指定了默认值，则新元素均为默认值（旧元素不变）

    • 时间复杂度：O(n)

  • 如果用普通数组 int mat[1000010][1000010]，浪费内存，会导致 MLE。
  • 如果使用 vector<vector<int>> mat(n + 10, vector<int> (m + 10))，完美解决该问题。
  • 另外，vector 的数据储存在堆空间中，不会爆栈。
  • 先指定长度，后赋值，会比一个个push_back快
  • vector 获取长度的方法 .size() 返回值类型为 size_t，通常 OJ 平台使用的是 32 位编译器（有些平台例如 cf 可选 64 位），那么该类型范围为[0,232)
    .
#include<iostream>
#include<vector>//注意头文件
using namespace std;

int main()
{
	//第一种方式
	vector <int> v;//定义数组,初始长度为0
	//第二种方式
	vector <int> v1(10, 2);//定义一个数组空间，全都初始化为2
	//第三种方式
	vector <int> v3(10);//相当于(10,0)，全初始化为0

	v.resize(10);//重新定义数组大小，初始化分配也都是0

	for (int i = 0; i < 10; i++)
	{
		v[i] = i;
	}
	//v.begin()指的是：第一个元素的索引	//v.end()指的是：最后一个元素的下一个位置
	for (auto p = v.begin(); p != v.end(); p++)//迭代器：不用改变循环，自动输出数组
	{
		cout << *p << " ";
	}
	cout << endl;

	v.push_back(11);//在末尾添加元素
	v.pop_back();//移除最后一个元素

	for (int i = 0; i < 11; i++)
	{
		cout << v[i] << " ";
	}
	cout << endl;
	cout << v.size() << endl;//获取数组长度
	return 0;
}

day05-STL_set(集合)

  • 无下标索引
  • 元素只读
  • 不能用迭代器计算下标
  • 互异性
  • 构造
    set<类型, 比较器> st
    set<int> st1 储存int的集合（从小到大）
    set<int, greater<int>> st2; 储存int的集合（从大到小）
  • 插入
    • .insert()
  • 删除
    • .erase()
  • 查找
    • .find()
  • 判断是否存在
    • .count()
#include<iostream>
#include<set>//注意头文件
using namespace std;
int main()
{
	//创建集合
	set <int> s;//和数组不同，后面不能带参数，集合具有互异性

	//处理
	s.insert(1);//插入
	s.insert(2);
	s.insert(3);

	for (auto p = s.begin(); p != s.end(); p++)//遍历
	{
		cout << *p << endl;
	}
	cout << endl;
	
	cout << (s.find(2) != s.end()) << endl;//查找
	//1 2 3 s.find()的返回值是个指针，如果不等于s.end(),说明找到了这个元素，上面一行代码输出1
	cout << (s.find(4) != s.end()) << endl;
	//没有找到4，这个元素，取相等，上面一行不成立，输出0

	s.erase(1);//删除集合中的1
	cout << (s.find(1) != s.end()) << endl;


	return 0;
}

day06-STL_map(键值对)

  • 互异性
  • 不能用迭代器计算下标
  • 构造
    map<键类型, 值类型, 比较器> mp
    map<int, int> mp1; int->int 的映射（键从小到大）
    map<int, int, greater<int>> st2; int->int 的映射（键从大到小）
  • 可以使用迭代器遍历
  • 查找
    • .find(元素)
  • 删除
    • .erase()
  • 判断是否存在
    • .count()
  • 场景
    • 统计字符串个数
#include<iostream>
#include<map>
using namespace std;
int main()
{
	map <string, int> m;
	m["hello"] = 2;//按照字符串第一位码表，第一位相同再看第二位...
	m["world"] = 3;
	m["ilove"] = 5;

	cout << "hello:" << m["hello"] << endl;

	for (auto p = m.begin(); p != m.end(); p++)
	{
		cout << p->first << ':' << p->second << endl;
	}

	cout << "map的长度为：" << m.size() << endl;
	return 0;
}
//map 它类似于一个结构体
struct s

day07-STL_stack(栈)

  • 先进后出
  • 不可访问
  • 进栈
    • .push(元素)
  • 出栈
    • .pop()
  • 取顶
    • .top()
#include<iostream>
#include<stack>
using namespace std;
int main()
{
	stack<int>s;
	s.push(1);//进栈
	s.push(2);
	s.push(3);
	s.pop();//出栈
	cout << s.top() << endl;//先进后出，后进先出
	cout << s.size() << endl;//获取大小
	//注意:栈不可以用迭代器
	return 0;
}

day08-STL_queue(队列)

  • 先进先出
  • 不可访问
  • 进队
    • .push(元素)
  • 出队
    • .pop()
  • 取队首
    • .front()
  • 取队尾
    • .back()
#include<iostream>
#include<queue>
using namespace std;
int main()
{
	queue<int>s;
	for (int i = 0; i < 10; i++)
	{
		s.push(i);
	}
	s.pop();//和栈不同，先进先出
	cout << "队首:" << s.front() << "队尾：" << s.back() << endl;
}

day08.1-priority_queue(优先队列)

  • 所有元素不可写
  • 仅堆顶可读
  • 构造
    • priority_queue<类型, 容器, 比较器> pque
  • 进堆
    • .push(元素)
  • 出堆
    • .pop()
  • 取堆顶
    • .top()
priority_queue<int>pque1;// 储存int的大顶堆
priority_queue<int,vector<int>,greater<int>> pque2; // 储存int的小顶堆

day09-STL_unordered_map和unordered_set

  • 没有顺序的键值对，和集合，可以减少时间，做题超时且没有顺序要求的时候可以用
#include<iostream>
#include<unordered_map>
#include<unordered_set>
using namespace std;
int main()
{
	unordered_map<string, int>m;
	unordered_set<int>s;

	s.insert(3);
	s.insert(5);
	s.insert(2);

	m["hello"] = 1;
	m["abc"] = 2;
	m["love"] = 3;

	for (auto p = s.begin(); p != s.end(); p++)
	{
		cout << *p << endl;
	}
	for (auto p = m.begin(); p != m.end(); p++)
	{
		cout << p->first <<" "<< p->second << endl;
	}
	return 0;
}
